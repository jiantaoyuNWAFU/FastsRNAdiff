# RegionRepCalc

## Introduction

This is a module for calculating Total Mapped Reads and Repeat-normalized total reads (Rep-total) for small RNA clusters.

## Authors

Xue S., Zhang X., Gao Y., etc.

## Version 

0.1.0

## Installation
```
1.Switch to the corresponding virtual environment

conda activate FastsRNAdiff

2.Switch to the tools directory

cd ~/tools

3.Installation

pip install -e .

4.Run

RegionRepCalc <BAM file> <sRNA cluster file> <output file>
```
For detailed instructions, please refer to README.md in the main directory.
